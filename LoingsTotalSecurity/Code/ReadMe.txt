使用：
	1. 启动程序请使用同目录内的Launcher.cmd，不要从root文件夹内直接启动；
	2. 虽然本程序开源，但源码禁止修改，避免影响使用；
	3. 设置内可以选择关闭实时防护或自我防护，它们默认是打开的；
	4. 不建议关闭主动防御，否则大部分功能将失去作用；
	5. 程序的自动更新附加到自我防护内，若关闭自我防护则禁用自动更新；
	6. 设置内可以手动更新产品和特征库；
	7. 卸载程序请以管理员身份运行root文件夹内的Uninst.cmd。

----------------------------------------------------------------------------------------------------
说明：
	1. 首次使用时建议到设置内手动检查更新。
	2. 不要使用自定义扫描自杀！
	3. 不要随便删文件！
	4. 不要随便更改文件名！
	5. 安装的目录尽量不要存在特殊字符。
	6. Administrator账户建议先打开UAC再打开主动防御，不打开UAC影响也不大，就是操作频繁一点。
