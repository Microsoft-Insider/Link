使用：
	1. 启动程序请使用同目录内的Launcher.cmd，不要从root内启动；
	2. 源码尽量不要改，可能看起来没作用的源码实际上作用挺大；
	3. 设置内可以选择关闭实时防护或自我防护，它们默认是打开的；
	4. 为了提高安全性，设置内可以选择打开主动防御，它默认是关闭的；
	5. 程序的自动更新附加到自我防护内，若关闭自我防护则禁用自动更新；
	6. 设置内可以手动更新产品和特征库；
	7. 卸载程序前建议先到设置内关闭其全部功能，再删除文件。

----------------------------------------------------------------------------------------------------
说明：
	1. 首次使用时建议到设置内手动检查更新。
	2. 不要使用自定义扫描自杀！
	3. 不要随便删文件！
	4. 不要随便更改文件名！
	5. 安装的目录尽量不要存在特殊字符。
	6. Administrator账户建议先打开UAC再打开主动防御，不打开UAC影响也不大，就是操作频繁一点。
